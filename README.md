UniPilot is a free, web-based application that helps students calculate and track their semester GPA and cumulative CGPA. This MVP is designed for personal use and feedback collection. A premium version with more advanced features and AI analytics will soon be in development.

---

## Features (MVP)

- Add courses with units and grades
- Automatic GPA & CGPA calculation
- Real-time updates
- Responsive UI for mobile and desktop

---

## Usage (For Students)

1. Open the web app in your browser.
2. Enter your course names, units, and grades.
3. GPA and cumulative CGPA are calculated automatically.
4. Refresh to reset or continue adding courses.

---

## Developer Setup

### Prerequisites
- Node.js 
- npm 

### Run Locally

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/UniPilot-MVP.git
cd GradeTrack-MVP
```

2. Install npm:
   ```bash
   npm install
   ```
3. Run it locally:
   ```bash
    npm run dev
   ```
4. Open http in browser
